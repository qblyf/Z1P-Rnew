# 智能匹配功能说明

## 功能概述

智能匹配功能允许用户输入商品名称，系统会自动在SKU数据库中查找最相似的商品，并返回匹配结果。

## 功能特点

1. **批量匹配**：支持一次输入多个商品名称（每行一个），批量进行匹配
2. **智能算法**：基于示例文件中的匹配算法，支持：
   - 品牌名称标准化（苹果→apple，华为→huawei等）
   - 关键词提取和匹配
   - 相似度计算
3. **结果展示**：清晰展示匹配状态、匹配的SKU、SPU、品牌和相似度
4. **数据导出**：支持将匹配结果导出为CSV文件

## 使用方法

1. 在导航栏点击"商品管理" → "智能匹配"
2. 在文本框中输入商品名称，每行一个，例如：
   ```
   华为 Mate 60 Pro 12+256 雅川青
   苹果 iPhone 15 Pro Max 256GB 钛金色
   小米14 Ultra 16GB+512GB 黑色
   ```
3. 点击"开始匹配"按钮
4. 查看匹配结果，可以导出为CSV文件

## 匹配算法

当前实现使用简化的匹配算法：

1. **字符串标准化**：
   - 转换为小写
   - 去除空格
   - 品牌名称标准化（中文→英文）

2. **相似度计算**：
   - 完全匹配：100%
   - 包含匹配：80%
   - 关键词匹配：根据匹配关键词数量计算

3. **匹配阈值**：默认60%，低于此阈值的不会被匹配

## 技术实现

- **前端框架**：React + Next.js
- **UI组件**：Ant Design
- **数据源**：通过`getSKUListJoinSPU` API获取SKU数据
- **匹配算法**：客户端实现，参考`.kiro/smartMatcher.ts`和`.kiro/matchingService.ts`
