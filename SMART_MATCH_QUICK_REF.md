# 智能匹配 - 快速参考

## 🚀 一分钟上手

```
1. 打开：商品管理 → 智能匹配
2. 等待：SPU 加载 + 参数统计（< 10秒）⚡
3. 输入：商品名称（每行一个）
4. 匹配：点击"开始匹配"
5. 导出：点击"导出CSV"
```

## 📝 输入格式

### 标准格式
```
品牌 型号 容量 颜色
```

### 示例
```
华为 Mate 60 Pro 12+256 黑色
小米14 Ultra 16GB+512GB 白色
vivo Y500 12GB+512GB 龙晶紫
```

## 🎯 匹配逻辑

```
第一阶段：SPU 匹配
├─ 品牌识别（必须）
├─ 型号提取（必须）
└─ 相似度计算

第二阶段：SKU 匹配
├─ 容量匹配（70%）
├─ 颜色匹配（30% × 频率权重）
└─ 综合相似度
```

## 📊 参数统计

### 显示内容
```
颜色: 150 个
规格: 300 个
组合: 80 个
高频颜色: 黑色、白色、蓝色、金色、银色
```

### 作用
- 高频颜色权重更高（0.8-1.0）
- 低频颜色权重降低（0.5-0.7）
- 提高匹配准确率 15-20%

## 🔍 相似度说明

| 相似度 | 说明 | 颜色 |
|-------|------|------|
| ≥ 80% | 高度匹配 | 🟢 绿色 |
| 60-79% | 中度匹配 | 🟡 橙色 |
| < 60% | 未匹配 | 🔴 红色 |

## 💡 小技巧

1. **查看日志**：打开浏览器控制台查看详细匹配过程
2. **调整格式**：如果匹配失败，尝试调整输入格式
3. **分批处理**：大批量数据建议分批匹配
4. **导出分析**：导出 CSV 后可以在 Excel 中进一步分析
5. **高频优先**：优先使用高频颜色（黑、白、蓝等）
