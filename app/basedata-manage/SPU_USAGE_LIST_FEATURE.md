# SPU使用列表功能

## 功能概述

在基础数据管理页面的规格编辑抽屉中，新增"使用列表"标签页，可以查看使用该规格的所有SPU。

## 使用方法

1. 进入"数据管理" → "基础数据管理"
2. 选择"版本管理"、"配置管理"或"颜色管理"标签页
3. 点击任意规格卡片，打开编辑抽屉
4. 在抽屉中切换到"使用列表"标签页
5. 查看使用该规格的所有SPU列表

## 功能特性

### 标签页结构
- **基本信息**：编辑规格的值、描述、排序权重
- **使用列表**：查看使用该规格的所有SPU（新增）

### 显示内容
- SPU总数统计（带蓝色徽章）
- SPU列表，每项包含：
  - SPU名称（加粗显示）
  - SPU ID（灰色小字）
  - SKU数量（蓝色标签）
- 列表支持滚动（最大高度500px）
- 悬停高亮效果

### 性能优化
- **懒加载**：仅在切换到"使用列表"标签页时才加载数据
- **缓存机制**：已加载的数据会缓存，避免重复请求
- **加载提示**：显示Spin加载状态
- **空状态**：无数据时显示友好的Empty组件

## 技术实现

### 关键代码
- 文件：`Z1P-Rnew/app/basedata-manage/page.tsx`
- 组件：`SpecEdit` - 规格编辑组件
- 函数：`loadSpuList()` - 加载SPU列表

### 数据流
1. 用户切换到"使用列表"标签页
2. 触发`loadSpuList()`函数
3. 查询所有"在用"状态的SPU
4. 过滤出包含该规格值的SPU
5. 显示在列表中

### 规格类型映射
- 版本管理（组合）：检查SKU的`combo`字段
- 配置管理（规格）：检查SKU的`spec`字段
- 颜色管理（颜色）：检查SKU的`color`字段

## 使用场景

1. **影响评估**：在修改或删除规格前，了解会影响哪些SPU
2. **数据审计**：检查规格的实际使用情况
3. **快速定位**：找到使用特定规格的所有产品
4. **清理决策**：判断是否可以安全删除未使用的规格

## 界面截图说明

### 标签页切换
- 默认显示"基本信息"标签页
- "使用列表"标签页标题显示SPU数量徽章

### 列表样式
- 每个SPU项占一行
- 左侧显示名称和ID
- 右侧显示SKU数量标签
- 项之间有分隔线
- 悬停时背景变为浅灰色

## 注意事项

1. 仅统计"在用"状态的SPU
2. 数据基于当前数据库状态
3. 大量数据时首次加载可能需要几秒钟
4. 列表最多显示10000个SPU（受API限制）
