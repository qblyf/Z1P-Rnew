# 品牌管理页面优化说明

## 优化目标
将品牌列表从每页10条优化为每页50条，提升空间利用率，减少翻页次数。

## 主要改动

### 1. 表格布局优化
- **尺寸**: 从 `size="middle"` 改为 `size="small"`
- **分页**: 默认每页从 10 条改为 50 条
- **分页选项**: `[50, 100, 200]` 支持更大数据量展示
- **滚动高度**: 从固定 400px 改为 `calc(100vh - 420px)` 自适应屏幕

### 2. 列宽优化
- 品牌名称: 180px → 160px (固定左侧)
- 拼音码: 120px → 100px
- 排序: 80px → 70px
- 状态: 100px → 80px
- 操作: 100px → 80px (固定右侧)

### 3. 样式紧凑化
```css
/* 单元格内边距 */
td: padding: 4px 8px (原 12px 16px)
th: padding: 8px 8px (原 16px 16px)

/* 行高 */
line-height: 1.3 (原 1.5)

/* 字体大小 */
标签: 13px (原 14px)
文本: 12px (原 13px)
```

### 4. 交互优化
- 添加行悬停效果
- 固定首列和操作列，方便横向滚动时查看
- 保持所有筛选、排序功能

## 效果预期

### 优化前
- 每页显示: 10 条
- 200 个品牌需要: 20 页
- 单行高度: ~48px

### 优化后
- 每页显示: 50 条
- 200 个品牌需要: 4 页
- 单行高度: ~28px
- **一屏可显示约 50-60 个品牌** (取决于屏幕高度)

## 兼容性
- 保持所有原有功能不变
- 响应式布局适配移动端
- 支持搜索、筛选、排序
